## 第一步：拥有一台自己的服务器（阿里云）

服务器本质上就是一台放在云端机房里的电脑。对于新手，我们强烈推荐购买**阿里云的“轻量应用服务器”**，它比传统的 ECS 云服务器更便宜、配置更简单。

1. **注册与登录**：前往 [阿里云官网](https://www.aliyun.com/) 注册账号并完成实名认证。
2. **找到产品**：搜索“轻量应用服务器”并点击购买。
3. **选择配置**：
   - **地域**：选择离你目标用户最近的城市（如“华东1 杭州”或“华北2 北京”）。
   - **镜像**：选择 **系统镜像 -> Ubuntu 22.04 或 24.04**（Ubuntu 是最适合新手的 Linux 系统，网上教程最多）。
   - **套餐配置**：本项目包含数据库和 AI 组件，建议至少选择 **2核 CPU / 4G 内存** 的套餐。
   - **购买时长**：按需选择（新用户通常有包年特惠）。
4. **获取公网 IP 和密码**：
   - 购买成功后，进入轻量应用服务器控制台。
   - 你会看到一个 **公网 IP 地址**（比如 `123.45.67.89`），这就是你服务器在互联网上的“门牌号”。
   - 在控制台找到“重置密码”选项，为 `root` 用户设置一个复杂的密码并记好。

## 第二步：连接服务器与基础安全设置

现在我们要通过你自己的电脑，远程操控这台远在机房的服务器。

### 1. 连接服务器 (SSH)
打开你电脑上的终端（Windows 用户打开 `cmd` 或 `PowerShell`，Mac 用户打开 `Terminal`），输入以下命令：

```bash
# 将 123.45.67.89 替换为你服务器的真实公网 IP
ssh root@123.45.67.89
```
首次连接会提示 `Are you sure you want to continue connecting?`，输入 `yes` 回车，然后输入你刚才设置的密码（输入密码时屏幕不会显示字符，这是正常的，输完直接回车）。

看到类似 `root@aliyun:~#` 的提示符，说明你已经成功进入服务器了！

### 2. 基础安全设置（防火墙）
为了防止黑客扫描，我们需要开启防火墙，**只允许特定的端口（通道）对外开放**。
在服务器终端依次输入以下命令：

```bash
# 允许 SSH 连接（极其重要，否则你下次就连不上服务器了！）
ufw allow 22

# 允许 HTTP 和 HTTPS 网页访问
ufw allow 80
ufw allow 443

# 允许我们项目的前端和后端端口
ufw allow 18080
ufw allow 18000

# 启用防火墙（输入 y 确认）
ufw enable
```

*注意：阿里云控制台也有一个“防火墙/安全组”设置，你需要去网页控制台把 `80`, `443`, `18080`, `18000` 这几个端口也添加进允许规则里。*

## 第三步：安装“装箱神器” Docker

**什么是 Docker？**
以前部署网站，你需要手动安装 Python、Node.js、数据库，经常遇到“在我电脑上能跑，在服务器上报错”的问题。Docker 就像集装箱，它把你的代码和运行环境打包在一起，保证在任何机器上都能完美运行。

在服务器终端运行以下命令，一键安装 Docker：

```bash
# 下载并运行官方安装脚本
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh

# 验证是否安装成功（如果输出版本号说明成功）
docker --version
```

## 第四步：把代码搬到服务器并运行

环境准备好了，现在把我们的项目跑起来。

### 1. 下载代码
```bash
# 安装 git 工具
apt update && apt install git -y

# 下载项目代码
git clone https://github.com/shadow-opt/SDU_Archive.git

# 进入项目文件夹
cd SDU_Archive
```

### 2. 配置环境变量
项目需要一些密钥才能运行。我们提供了一个模板文件 `.env.example`。

```bash
# 复制一份模板文件并命名为 .env
cp .env.example .env

# 使用 nano 编辑器打开它
nano .env
```

在打开的编辑器中，你需要修改以下内容：
1. `SECRET_KEY`：随便敲一串复杂的英文字母和数字（用于加密用户登录状态）。
2. `OPENAI_API_KEY`：填入你的 OpenAI API Key（如果暂时没有，可以先留空，但 AI 功能将无法使用）。

修改完成后，按 `Ctrl + O` 保存，回车确认，然后按 `Ctrl + X` 退出编辑器。

### 3. 一键启动！
见证奇迹的时刻到了，输入以下命令：

```bash
docker compose up -d
```

*解释：`up` 表示启动，`-d` 表示在后台运行。Docker 会自动去下载数据库、构建前端和后端，这可能需要几分钟时间，请耐心等待。*

当所有进度条跑完，并在终端看到绿色的 `Started` 字样时，恭喜你，网站部署成功了！

### 4. 访问你的网站
打开你电脑的浏览器，输入：
`http://你的服务器公网IP:18080`

你应该能看到 SDU Archive 的登录界面了！
- **默认管理员账号**: `admin@example.com`
- **默认密码**: `admin12345678`
*(登录后请务必在系统中修改密码)*

## 第五步：绑定域名与开启 HTTPS（进阶）

现在你的网站只能通过 IP 地址和端口号访问，看起来不太正式。如果你想让它变成 `https://www.yourname.com`，你需要做以下几步：

1. **买域名**：在阿里云的“域名注册”页面买一个你喜欢的域名。
2. **实名认证与备案**：国内服务器的域名必须经过工信部备案才能访问（阿里云控制台有详细的备案指引，通常需要几天时间）。
3. **解析域名**：在阿里云的“云解析 DNS”中，添加一条 `A记录`，将你的域名指向服务器的公网 IP。
4. **配置反向代理（Nginx）**：
   为了去掉网址后面的 `:18080` 并加上安全锁（HTTPS），最适合新手的方法是使用 **Nginx Proxy Manager**。它提供了一个可视化的网页界面，让你点点鼠标就能配置好域名和免费的 SSL 证书。
   *(由于篇幅限制，你可以搜索“Nginx Proxy Manager 部署教程”来完成这最后一步的升华。)*
